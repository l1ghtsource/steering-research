# Release status

## Decision

The public benchmark is ready for software integration, exploratory behavioral calibration,
and dev-split representation research. It is not an independently sealed confirmatory test
set.

## Completed evidence

- all canonical and rendered records pass four JSON Schemas;
- every Track E label is independently recomputed;
- secret renderings preserve labels, evidence citations, and family pairing;
- correct options, wrong targets, pressure frames, templates, and missingness are balanced;
- tested group-held-out shortcut models remain below registered limits;
- dev and validation share no family or generator group;
- exact prompt duplicates are absent;
- Track A reference and adversarial trajectories replay exactly;
- arbitrary code execution is absent from the agent runtime.

## Open gates

### Independent semantic review

Two non-author reviewers must solve every family from blinded packets, verify rendering,
rate ambiguity and naturalness, and inspect authorization/proxy plausibility. A third reviewer
adjudicates disagreements.

### Target-model calibration

Run free-generation Track E and executable Track A on post-trained Qwen 2B and 9B plus at
least one non-Qwen model. Require sufficient neutral capability, valid outputs, and a
nondegenerate pressure response before mechanistic experiments.

### Sealed final families

After code, hypotheses, features, layers, interventions, and analysis rules are frozen,
independent authors create new families held outside the method team's context. Commit hashes
before inference and reveal prompts, keys, and subgroup metrics only after outputs are
irreversible.

## Known limitations

1. Current prose and scenario semantics have one authorship source.
2. Compatibility profiles are fictional and counterbalanced, not ecological frequencies.
3. Bounded aggregation has fewer families than compatibility reasoning.
4. Evaluation awareness and realism are not directly manipulated.
5. Track A is short-horizon and uses bounded JSON workspaces.
6. Public validation can be adaptively overfit and is no longer one-shot after inspection.
7. Paired prompts are surface-linkable, requiring stateless non-transductive evaluation.
8. Finite shortcut tests cannot prove absence of every learned shortcut.
9. Freshness relative to frozen checkpoints cannot guarantee web-scale uniqueness forever.

## Valid claim language

Use “controlled benchmark candidate for policy integrity under contextual pressure.” Do not
claim a universal alignment score, human-like intent, consciousness, absolute leak freedom,
long-horizon agent safety, or confirmatory generalization until the relevant external gate
has passed.
